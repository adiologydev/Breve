# Breve
A Simple Link Shortening Service
</div>

## Getting Started
### 1. Clone the repository
```bash
git clone https://github.com/AdityaTD/Breve.git
```

### 2. Install dependencies
```bash
cd Breve
npm install
```

### 3. Fill in the environment variables
```bash
cp .env.example .env
```

Edit the `.env` file and enter your variables.

### 4. Start the server
```bash
npm start
```
